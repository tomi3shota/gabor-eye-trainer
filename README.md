# ガボールパッチ・アイ・トレーナー

ガボールパッチ（ガウス包絡×正弦波格子のパターン画像）を使った視力トレーニングゲームです。
お題として提示された画像と同じパターンが、9個の画像の中にあるかどうかを判断します。

PC版に加えて、スマートフォン＋カードボード型VRゴーグルで遊べるVR版も用意されており、
提示環境（画面 or VR）による効果の違いを比較できるようになっています。

ガボールパッチは毎回`gabor-canvas.js`でブラウザ上に動的生成しており、固定の画像セットを
使い回さない（参加者が画像そのものを覚えて正答率が上がってしまうのを防ぐ）設計になっています。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | トップページ（難易度選択） |
| `game.html` / `script.js` | PC版ゲーム画面とロジック |
| `vr-index.html` | VR版の準備ページ（センサー許可・難易度選択） |
| `vr-game.html` / `vr-script.js` | VR版ゲーム画面とロジック（Three.js使用） |
| `gabor-canvas.js` | ガボールパッチをCanvas上で動的生成する共通モジュール（PC/VR共通） |
| `score-export.js` | ゲーム終了時にスコアをGoogleフォームへ自動送信する共通モジュール |
| `results.html` | スコア履歴・統計グラフ（Chart.js使用） |
| `style.css` | 共通スタイル |
| `analysis/` | 発表用のスコア分析パイプライン（詳細は`analysis/README.md`） |
| `images/`, `generate_gabor.py` | 旧バージョンで使っていた静的なガボールパッチ画像・生成スクリプト（現在のゲームは使用していない） |

## 遊び方

1. `index.html` をブラウザで開く
2. 難易度（初級・中級・上級）を選んでゲーム開始
3. お題の画像と同じパターンが9個の画像の中にあれば「ある！」、なければ「ない！」をクリック
4. 10問で1セット完了。結果は `results.html` からスコア履歴として確認可能

VR版で遊ぶ場合は、スマートフォンで `vr-index.html` を開き、画面の案内に従ってセンサーの使用を許可してください。
